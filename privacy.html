Privacy Policy – PGx Clinical Decision Support Application

Effective Date: 16 April 2026

The PGx Clinical Decision Support (CDS) application (“Application”) is designed to support healthcare professionals by providing pharmacogenomic (PGx)-based advisory information.

1. Information We Collect

The Application collects limited user data required for functionality:

  • Account Information: Email address and login credentials
  • Usage Data: Drug searches and interaction logs
  • Technical Data: Device and application performance data

The Application is designed to minimize the collection of personally identifiable patient data.

2. Use of Information

Collected data is used for:

  • Enabling core application functionality
  • Providing clinical decision support outputs
  • Improving system performance and usability
  • Ensuring system security

3. Health-Related Data

The Application may process pharmacogenomic inputs (e.g., genotype information) entered by users for real-time advisory output.

Such data:

  • Is used only for immediate processing
  • Is not intended to be stored as identifiable patient data unless explicitly configured

4. Data Sharing

The Application does not sell, rent, or share user data with third parties for marketing purposes.

Data may only be processed within secure infrastructure required for application functionality.

5. Data Security

Reasonable technical and organizational measures are implemented to protect data, including:

  • Encrypted communication (HTTPS)
  • Access controls
  • Secure storage practices

6. Data Retention

Data is retained only as necessary for operational and analytical purposes, in accordance with applicable policies.

7. User Rights

Users may request:

  • Access to their data
  • Correction of inaccuracies
  • Deletion of account data

8. Intended Users

This Application is intended for use by qualified healthcare professionals only.

9. Changes to Policy

This policy may be updated periodically. Continued use of the Application constitutes acceptance of updated terms.

10. Contact
